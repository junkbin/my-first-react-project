# Project Setup

- https://nodejs.org/en
- https://vite.dev/guide/

## Basic Commands

```
node -v
npm -v
```

## Create Project

```
npm create vite
- And follow the instruction
```

## Install the modules

```
npm install
- After this you can see `node_modules` folder in the project explorer
```

## Run the Project

```
npm run dev
```
